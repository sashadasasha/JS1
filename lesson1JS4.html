/* И async, и defer позволяют браузеру (дают ему такую инструкцию) загрузить код JavaScript, не приостанавливая при этом загрузку html страницы. При атрибуте Async браузер продолжает загрузку html страницы,одновременно начиная загрузку js файла, когда доходит до него. После того, как браузер загрузит js файл, загрузка основного html приостановится и начнет выполняться загрузившийся сценарий js файла. Defer позволит браузеру сначала полностью загрузить html страницу и лишь затем начать выаолнение js кода. При этом его загрузка идет параллельно с загрузкой html-документа. */
